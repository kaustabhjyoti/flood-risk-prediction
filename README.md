# Flood Risk Prediction Using Random Forest

## About the Project :-

This project predicts flood risk levels (High, Medium, or Low) using environmental factors such as rainfall, elevation, and slope.

The main objective was to understand how classification models work and how machine learning can be applied to environmental problems.

## What I Did :-

* Created a flood risk dataset
* Selected relevant environmental features
* Split the data into training and testing sets
* Trained a Random Forest Classifier
* Generated predictions
* Evaluated model accuracy
* Analyzed feature importance

## Technologies Used :-

* Python
* Pandas
* Scikit-Learn

## What I Learned :-

Through this project, I learned:

* Classification problems
* Random Forest algorithm
* Accuracy score
* Confusion Matrix
* Feature Importance

## Future Improvements :-

I would like to work with real-world flood datasets and include additional factors such as river distance and land use.
